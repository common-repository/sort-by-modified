Plugin adds a sortable modified date column to posts & pages within wordpress.

== Description ==

Adds a sortable modified date column to posts, pages, and custom post types.

== Installation ==

1. Upload the `sort_by_modified` directory to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
